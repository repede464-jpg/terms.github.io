# terms.github.io

Royden Video Uploader processes only the data required to publish videos to connected social media accounts: video files, captions, publishing status, and OAuth access tokens. The tool does not sell personal data. OAuth tokens are used only to publish content authorized by the account owner. Users may revoke access at any time through the relevant platform account settings.
